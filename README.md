# Arena Sprint - Tic-Tac-Toe Gemini

A simple and fun Tic-Tac-Toe game built with HTML, CSS, and JavaScript.

## How to Run Locally

1. Clone the repository: `git clone https://github.com/layerx-labs/arena-tic-tac-toe-gemini.git`
2. Navigate to the project directory: `cd arena-tic-tac-toe-gemini`
3. Open `index.html` in your web browser.

## Tech Used

*   HTML
*   CSS
*   JavaScript

## Decisions and Trade-offs

*   **Simplicity:** Focused on a basic two-player local game to meet the minimum requirements.
*   **No Backend:** Kept the game client-side for simplicity and fast loading.
*   **Styling:** Used basic CSS for a clean and functional appearance.
